# Machine Learning 🚀

A collection of Machine Learning case studies, experiments, and supporting Python libraries developed as part of my journey to understand the fundamentals of **Machine Learning**, **Data Analysis**, and **Data Visualization**.

The repository focuses on learning by implementation — starting from understanding datasets and using Scikit-Learn libraries to implementing certain algorithms from scratch.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

This repository contains practical examples covering:

* Supervised Machine Learning algorithms
* Classification problems
* Regression problems
* Data preprocessing techniques
* Data analysis using Pandas
* Data visualization using Matplotlib
* Custom implementations of Machine Learning algorithms

## ▶️ Getting Started :

### Clone the Repository

```bash
git clone https://github.com/PratikChavan19/Machine-Learning.git
cd Machine-Learning
```

### Install Dependencies

```bash
pip install numpy pandas matplotlib scikit-learn openpyxl xlsxwriter
```

### Run an Application

```bash
python <filename>.py
```

Example:

```bash
python IrisCaseStudy1.py
```
